### 1.7 [字典](Basic_python_syntax/Dictionary)

字典是一种比较特别的数据类型，字典中每个成员以“键:值”对的形式成对存在。字典与列表相比，最大的不同在于字典是无序的，其成员位置只是象征性的。在字典中通过键来访问成员，而不能通过其位置来访问该成员。

键值可以取任何数据类型。
